# 树人健身房 AI 会员服务智能体 V1.0

面向浙江树人学院杨汛桥校区健身房会员的轻量网页应用。

## 功能

- 智能咨询：支持淋浴间、套餐价格、入会信息、反馈入口等常见问答。
- 套餐推荐：根据预计训练月数、每周训练次数、预算偏好和训练目标推荐年卡或月卡。
- 入会登记：记录学号、专业班级姓名、电话号码、套餐档位和备注。
- 会员管理：本地保存会员数据，支持搜索、删除和生成 Excel 登记表。
- 用户反馈：记录反馈类型、满意度和反馈内容，并生成 Word 统计文件。

## 套餐档位

- 年卡：365 元
- 月卡：200 元

## 使用方式

直接打开 `index.html` 即可使用。数据会保存在当前浏览器的本地存储中。

如需本地服务器预览，可在本目录运行：

```bash
python3 -m http.server 8000
```

然后访问 `http://localhost:8000`。

## 发布成微信可打开的网址

本项目是纯静态网站，可以部署到 Vercel、Netlify、GitHub Pages 或学校服务器。上线后得到的 `https://...` 地址可以直接发到微信打开。

推荐发布方式：

1. Vercel：新建项目，导入本目录或上传这些文件，框架选择 `Other`，输出目录保持默认。
2. Netlify：选择 `Deploy manually`，将整个项目文件夹拖入部署区域。
3. GitHub Pages：把文件推送到仓库，开启 Pages，发布分支选择 `main`，目录选择根目录。

注意：会员和反馈数据当前保存在访问者浏览器本地。如果需要多人共用同一套后台数据，需要再接入云数据库或表单后台。
